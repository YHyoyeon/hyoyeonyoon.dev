## 🔧 주요 트러블슈팅

### 1. 서브도메인 간 인증 문제 해결
- **문제**: dev2.auth.softnyx.com과 dev2.www.softnyx.com은 서브도메인이 달라서 로컬스토리지 공유 불가
- **해결**: 쿠키 기반 인증으로 전환하여 서브도메인 간 인증 문제 완전 해결
- **결과**: 사용자 경험 개선 및 안정적인 인증 시스템 구축

### 2. CORS 문제 해결
- **문제**: 마켓플레이스에서 image2 도메인 연결 시 CORS 에러 발생
- **해결**: CloudFront Distribution 설정 확인, API Gateway CORS 정책 수정
- **결과**: CORS 문제 완전 해결, image2 도메인 정상 연결 확인

### 3. Status API 버그 수정
- **문제**: 로그인 상태 확인 API 버그 및 로그아웃 기능 문제
- **해결**: 세션 쿠키 기반으로 데이터 처리하도록 변경
- **결과**: 안정적인 인증 상태 관리 구현

### 4. 회원가입 ID 유효성 검사 오류
- **문제**: 올바른 형식의 userId를 보냈는데도 400 에러 발생
- **해결**: ID 검증 로직 수정
- **결과**: 회원가입 정상 작동
